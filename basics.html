The Mailing List

The Speak Up! Mailing list is graciously hosted by the Python Software Foundation. The name of the mailing list is "talk-mentorship [at] python.org", the subscription interface is here: talk-mentorship. This mailing list adheres to the Code of Conduct (of course).

Please note that the list is locked so that archives are only available to subscribers, and non subscriber emails sent to the list must be approved by a moderator.

IRC

We have an IRC channel:

  • Server: irc.freenode.com
  • Channel: #speakup!
This IRC Channel adheres to the Code of Conduct (of course).
